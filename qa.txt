Обучался в Яндекс Практикуме на курсе Инженер по тестированию ПО
Курс состоит из 4 проектов и диплома(который включает в себя проверку всех необходимых навыков), сделанные мною проекты прилагаю

После 1го проекта научился:
  - Анализировать и декомпозировать требования
  - Выделять классы эквивалентности и граничные значения
  - Составлять тест-кейсы
  
После 2го проекта научился:
  - Выбирать конфигурации для кроссбраузерного тестирования
  - Составлять чек-листы, чтобы покрыть все проверки
  - Тестировать по своим тестам
  - Заводить баг репорты
  - Работать с Charles
  
После 3го проекта научился:
  - Тестировать мобильную версию веб-приложения
  - Тестировать мобильное приложение на Android Studio и на реальном устройстве
  - Составлять тесты к API
  - Тестировать API с помощью Postman
  
После 4го проекта научился:
  - Работать в консоли
  - Подключаться и работать с БД
  - Составлять несложные запросы в SQL
  - Работать с логами
  
  Тестировал учебный сервис Яндекс.Маршруты, Яндекс.Прилавок, а так же мобильное приложение Яндекс.Метро
